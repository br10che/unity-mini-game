# Mini Unity Game Prototype

This is a small 2D game prototype built in Unity as part of my introduction to game development.

## Features
- Player movement using keyboard input
- Collectable objects (coins)
- Score system
- Win condition when all coins are collected

## Purpose
This project was created to practice core gameplay mechanics, scripting in C#, and understanding Unity’s component-based system.

## Technologies
- Unity (2D)
- C#
